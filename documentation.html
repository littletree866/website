Geometric Runner (Runner)

Endless runner / jumper with shape-matching obstacles, power-ups and procedural background layers.

  • File: newgames/runner.html
  • Controls: Space to jump; keys 1/2/3 or automatic shape cycling
  • Features:
    • Parallax starfield / mountains / clouds background for improved depth.
    • Pause on tab hide to avoid timing drift and accidental score inflation.
    • Improved obstacle spawn spacing and cap to avoid memory growth.
    • Particles, gradient player drawing and simple sound effects (jump/collect/die/shape).
    • Best-run persistence (localStorage) and a visible restart button on death.

Play Geometric Dash

Empire BETA

Turn-based skirmish/adventure demo with regions, shop, loot and progressive enemies.

  • File: newgames/empire.html
  • Controls: mouse (click areas), shop interactions via prompts
  • Features:
    • Shop items now carry explicit type (weapon / armor) and purchases equip immediately.
    • Unlock checks (areas/regions) consider equipped weapon/armor stats so progression is intuitive.
    • Purchases and player state persist via localStorage; gold/loot animations and experience gain added.
    • Improved battle flow and basic feedback (shake/flash/gold popups).
