# Contributing to Infinity Nexus (littletree866.github.io)

Thanks for your interest in helping improve the site! Your contributions, either big or small are always welcome.

## How to Contribute

### Report Issues
- Found a bug, typo, or broken link? Create a pull request and I'll see if it works and add it in.
- Include screenshots if needed. I would like to see the issue, so be as descriptive as possible.

### Suggest Features
- Open the "disccusions" tab and post a message, I'll read it as soon as I can.
- You can also use this form: https://forms.gle/X8p7jkC74MFG9Evj7

### Coding & Content 
- Follow existing HTML, CSS, and JavaScript style.
- Keep everything organized and readable.
- Make sure it is in the same theme or similar theme

### Be Respectful
- Keep discussions polite and constructive.
- Feedback should focus on improving the site, I'll check your code to make sure it's good.
